== Description ==

Adds page numbers at the bottom for easier navigation. This is also better for seo as it will not take as many jumps to get to older pages for the search spiders. NO theme editing required.


== Installation ==

1. Upload `page-numbers-page-navigation.zip` to the `/wp-content/plugins/` directory and unzip
2. Activate the plugin through the 'Plugins' menu in WordPress
